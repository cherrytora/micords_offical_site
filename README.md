# Micords 官方網站
## 介紹

飲食、體重、血檢、用藥，一個 APP 全部記好。看診時快速查閱日常紀錄，和獸醫溝通不再手忙腳亂，在家正確掌握血檢數據與趨勢，每一步都有跡可循，奴才更安心。

## 品牌故事

### 為什麼做 Micords？

事情的一開始是先發現橘貓拉拉甲亢，結果過沒兩天，也發現白貓利利不吃不喝，帶去檢查才發現胰臟炎和慢性腎病2期，剛好兩隻貓幾乎同時發病，也同時都需要長期追蹤。

那段好幾家醫院、診所來回跑的過程，因為個想檢查數值、基礎數值、正常數值都不一樣，變成在就醫的過程前後都要不斷的翻筆記、回想之前數值和醫生說過什麼。

檢查報告出來之後，也沒辦法馬上確定這幾個禮拜下來的回診，數值是變好了嗎？還是其實沒有？尤其兩隻貓咪回診時間在周間差不到兩天，變成醫生問問題的時候會記錯貓咪，給出一個連醫生都會狐疑的答案。

那種手忙腳亂的感覺，讓我們決定做一個工具：「把對貓咪的愛，變成有跡可循的健康紀錄」。無論是想記錄日常或是需要看病紀錄醫療數據，都能透過Micords輕鬆完成。

Micords 由兩隻貓的奴才和被她血緣壓制的弟弟一起打造，分享給每一位照顧毛孩的爸媽。

貓咪紀錄，就找Micords。
